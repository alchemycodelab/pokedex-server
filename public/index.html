Pokedex API

There is one API endpoint: 

    
    https://alchemy-pokedex.herokuapp.com/api/pokedex

Paging

You can specify page and perPage via querystring. Defaults to first page and 20 pokemon per page. 

    
    https://alchemy-pokedex.herokuapp.com/api/pokedex?page=3&perPage=50

Querying

You can specify any object property as a query. Strings match if they are included in the specified property, numbers properties match if they are greater than or equal to the supplied value. 

    
    https://alchemy-pokedex.herokuapp.com/api/pokedex?pokemon=char&defense=50

Data Format

Return format is as follows: 

    
    {
        "count": 801,
        "page": 1,
        "perPage": 20,
        "results": [
            // pokemon objects
        ]
    }
